    # Vue+ts 实践

    # 为什么要使用Typescript

    • 类型约束能更好的帮我们发现bug，并且能得到更好的ide智能提示，提升开发效率
    • 从js过渡到 Ts 的成本很低
    • Ts 未来会大火，从主流的大型库用ts重写（包括vuejs）可以看出来，未来是一门必须要掌握的技术
